# PocketBach - by Jesse Lew

## A program that records a melody, transcribes it, analyzes it, and composes a fugue with it.


**Audio Recorder** - *Complete*

To record:
```
$ python3 audio_recorder.py
```

To play back:
```
$ aplay melody.wav
```
